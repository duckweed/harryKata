


1. Download JsTestDriver, and put jar in lib dir.

2. Run the following command to start a test server

java -jar lib\JsTestDriver-1.3.5.jar --port 9876 --browser "c:\Program Files (x86)\Google\Chrome\Application\chrome.exe"

3. Run the following command to run the actual test

java -jar lib\JsTestDriver-1.3.5.jar --tests all
